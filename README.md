# GameRecommender
A console-based video game recommender using Python for data preprocessing and Java for the application logic

The original data can be found here:
https://huggingface.co/datasets/FronkonGames/steam-games-dataset/tree/main

I downloaded the json file, then cleaned up the file only taking variables I needed using a python script. 
The raw and cleaned data are not part of the repository as they were too large to add.
